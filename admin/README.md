# SSMI Admin

This folder is the future admin web app for the church site.

## Intended structure

- `src/app`: shared admin data and route maps.
- `src/components`: dashboard layout and reusable controls.
- `src/pages`: content management, branch management, media, and settings views.

## Why this exists

The FlutterFlow app contains a lot of church content that needs a better editing surface.
This admin app will manage those pages, branches, ministries, and media independently from
the public website.
